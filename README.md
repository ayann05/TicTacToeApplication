# Tic Tac Toe Game 
This is a classic Tic Tac Toe game application. The game allows two players to take turns marking spaces in a 3x3 grid with their respective symbols (usually 'X' and 'O').

Features
Two-Player Mode: Play against another person on the same device.
Simple and Intuitive Interface: Easy-to-use interface for a seamless gaming experience.
Real-Time Game Updates: Instantly see the game board updates as players make their moves.
Win Detection: Automatically detects winning combinations and declares the winner or a draw.
